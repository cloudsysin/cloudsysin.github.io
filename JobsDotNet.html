职业描述

岗位职责：

  • 参与公司内部业务系统的分析、设计工作,负责Project中主要功能的代码实现, 解决Project中的关键问题和技术难题；
  • 协助项目经理、开发经理保证项目进度和质量。

任职要求:

  • 计算机相关专业，大学专科以上学历；
  • 3年以上的软件开发经验，具备2年以上基于.NET Platform的开发经验；
  • 熟练掌握C#语言, 具备.Net Framework 1.1, 2.0, 3.5相关开发经验；
  • 熟练掌握Ado.net, 了解并掌握业界主流的.Net平台的O/R Maping组件；
  • 熟练掌握Asp.net Web Form相关开发. 熟悉Web前端相关开发技术, Html, Css, Javascript；
  • 熟练掌握1种或多种主流Javascript Framework(JQuery, Prototype, ExtJs, etc);
  • 熟练掌握.net WinForm相关开发;
  • 熟悉1种或多种.net平台下的分布式技术(Remoting, WebService, WCF, etc);
  • 深入理解面向对象设计原则, 具备良好的OO设计和开发能力;
  • 熟练掌握SQL Server数据库相关开发, 具备良好的SQL Script编写、优化、分析能力. 了解并掌握基本数据库设计原则;
  • 具有良好的英语读写能力;
  • 具备良好的沟通和解决问题的能力,能够在较大的压力下工作, 具备良好的团队协作精神;
  • 具备以下能力者, 公司将会优先考虑:
  • (1).熟练掌握silverlight, Share Point, Asp.net MVC, WCF, WPF, WF, Biztalk, MSMQ, 具备多个基于上述技术的大型企业级应用项目的相关开发经验;
  • (2).具有大型分布式系统或大型前台网站的设计和实施经验, 对可伸缩性, 性能, 可扩展性, 可维护性, 可重用性有较好的理解, 能够根据实际应用场景对各项要求进行平衡。
